About Modern Scheduler

Welcome to Modern Scheduler, your ultimate task and event management tool!

Functionality

Modern Scheduler allows you to:

  • Manage your tasks by adding, updating, and deleting them.
  • View your tasks in a convenient table format.
  • Organize your tasks by status (To Do, In Progress, Done, On Hold).
  • Set due dates for your tasks and view them on the calendar.
  • View and manage your events by adding, updating, and deleting them.
  • See all your tasks and events in a calendar view with the ability to change the month.
  • Update your profile information, including your display name and password.
  • Toggle dark mode for a better viewing experience.
  • View login/logout and task history (admin only).

Implementation Details

Modern Scheduler is implemented using Python's Flask framework for the backend and SQLite for the database. Here's how some of the key features are implemented:

  • Tasks: Users can add, update, and delete tasks. Task status is managed using predefined options (To Do, In Progress, Done, On Hold).
  • Events: Users can add, update, and delete events. Events are associated with a specific date and can have optional comments.
  • Calendar: The calendar view displays all tasks and events for a selected month. Tasks are color-coded based on their status.
  • User Authentication: Users can register, log in, and log out. Passwords are hashed using bcrypt for security.
  • Profile Management: Users can update their display name and password.
  • Dark Mode: Users can toggle dark mode for a different visual theme.
  • History: Admin users can view login/logout and task history.

Usage Examples

Here are some examples of how you can use Modern Scheduler:

  • Adding a Task: Navigate to the "Tasks" page and click "Add Task". Enter the task details and click "Submit".
  • Updating Task Status: On the "Tasks" page, click the dropdown next to a task and select a new status.
  • Viewing Calendar: Navigate to the "Calendar" page to see all your tasks and events in a monthly view. You can navigate between months using the arrow buttons.
  • Changing Profile Information: Go to the "Profile" page to update your display name or password.
  • Logging Out: Click on "Logout" in the navigation bar to log out of your account.
